# 🕸️ Web Scraping com Python

## 📌 Descrição do Projeto

Este repositório contém um projeto de **Web Scraping em Python**, desenvolvido com o objetivo de aprendizado prático na coleta de dados da web.

O projeto foi construído **em etapas**, com base em um tutorial do YouTube, permitindo compreender de forma progressiva todo o processo — desde requisições básicas até a extração e organização dos dados.

---

## 🚀 Objetivos

- Aprender os fundamentos de Web Scraping com Python  
- Entender a estrutura de páginas HTML  
- Automatizar a coleta de dados  
- Organizar e tratar as informações extraídas  
- Desenvolver um projeto passo a passo para melhor aprendizado  

---

## 🧠 Tecnologias Utilizadas

- Python  
- requests  
- BeautifulSoup  

---

## 📂 Estrutura do Projeto

O projeto está dividido em etapas para facilitar o aprendizado:

etapa1/ → primeiras requisições
etapa2/ → parsing de HTML
etapa3/ → extração de dados
etapa4/ → tratamento e organização

Cada pasta representa uma evolução no processo de scraping.

---
